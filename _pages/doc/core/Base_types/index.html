Module Base_types

Base FIX types.

type fix_string =
| Model_string of int
| Admin_string of int
| Empty_string

We encode the strings as unique 1-to-1 integers. The strings that are only relevant for administrative messages are encoded by hashing the string. Codes for the business-logic relevant strings are generated by the DSL compiler.

type fix_symbol = fix_string

Symbol is a type alias for fix_string
